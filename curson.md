# CV Project

Personal CV/resume website for Ashraf Ahmed, served from XAMPP at `d:\xampp\htdocs\cv`.

## Structure

- `index.html` — Main CV page (Bootstrap 4, Font Awesome, custom CSS)
- `CSS/style.css` — Primary styles
- `CSS/responsive.css` — Responsive layout rules
- `JS/main.js` — Main site scripts
- `practice.html` — JavaScript practice sandbox
- `JS/practice.js` — Practice scripts
- `*Array.html`, `ForEachFunction.html`, `ObjectLoop.html` — Standalone JS learning pages

## Stack

- HTML5
- CSS3 (custom + Bootstrap 4 CDN)
- Vanilla JavaScript (no build step or framework)

## Conventions

- Keep changes minimal and focused on the requested file or feature.
- Match existing HTML/CSS/JS style in nearby files.
- Practice and learning files are separate from the main CV site; avoid mixing them into `index.html` unless asked.

## Local development

Open via XAMPP (Apache) or directly in a browser:

```
http://localhost/cv/
```
